# GătitSimplu
Public domain recips in Romanian. Largely inspired by [based.cooking](https://github.com/lukesmithxyz/based.cooking).

Site conținând rețete în domeniul public în limba română. Inspirat de [based.cooking](https://github.com/lukesmithxyz/based.cooking).

### Philosophy / Filosofie

Recipes should be simple enough to follow without an image and use simple vocabulary. Translated recipes should not be written on a mot-a-mot basis, instead opting for clarity.
